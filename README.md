 ### Features
 - ToDO (add, remove and check)
 - Third Api (post and get)
 - Maps( get User Location)

 ## Build Setup

 - Install dependencies
 ```bash
 # (Yarn)
 $ yarn install
 # (npm)
 $ npm install
 ```
 - Run
 ```bash
  $ npx react-native run-android
 ```
